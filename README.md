## PENJELASAN SINGKAT TENTANG FUNGSI DAN CARA KERJA PROGRAM
- Program meminta input : berat paket,jarak pengiriman,panjang,lebar,dan tinggi paket
- Menghitung volume = panjang x lebar x tinggi
- Menentukan biaya per kg berdasarkan jarak:
  <10 km = Rp 4250/kg
  10 km = Rp 6000/kg
- jika volume > 100 cm , tambahkan biaya Rp 50.000.
- Total biaya =( berat x biaya per kg) + biaya volume(jika ada).
- Hasil ditampilkan ke pengguna
